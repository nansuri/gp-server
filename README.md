# GP - General Purpose Server

GP Server is a simple Golang based server which serve many purposes #eaa 

## First Time User

- Make sure you already setup golang on your local, can use this tutorial if you need `https://golang.org/doc/install`
- Clone this repository and place it under `go/src/github.com/nansuri/gp-server`
- Try to run the app first using `go run main.go`
- Allow the port to be accesed
- Open your postman and hit this example API
```sh
http://localhost:3025/user/ping

Request: 
{
    "first_name": "test"
}
```
- Voilla you done your first run!! Happy Coding!
